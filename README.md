# Travel-Agency
Small Python 3 Program utilizing sockets (TCP) and multi-threading, a Travel Agency.
This project implements all of the commands specified in the given list.
Every command should work correctly as you will see on the video.
The host used is localhost and port used is 10000.


LINK FOR THE VIDEO : https://youtu.be/yik-GTIopTY

Command list
LIST/l
SEARCHD DEST/sd
SEARCHALL DEST/all
SEARCHS DEPARTURE/sdp
BUY_TICKETS/b
    - RETURN_TICKET
    -RETURNRT_TICKET
